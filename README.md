# ❤️ AI Influence on Healthcare: Heart Disease Prediction

## 📌 Overview
This project focuses on predicting heart disease using Machine Learning techniques. The goal is to assist in early diagnosis by analyzing patient health data and improving clinical decision-making.

Heart disease remains one of the leading causes of death worldwide, and early detection can significantly reduce risks. This project leverages data-driven models to enhance prediction accuracy.

---

## 🚀 Features
- Data preprocessing (handling missing values, normalization, outlier removal)
- Feature selection using correlation analysis & RFE
- Implementation of multiple ML algorithms:
  - Logistic Regression
  - Support Vector Machine (SVM)
  - Random Forest
  - Gradient Boosting
  - K-Nearest Neighbors (KNN)
- Model evaluation using:
  - Accuracy
  - Precision
  - Recall
  - F1 Score
- Comparison of model performance

---

## 📊 Dataset
- Dataset contains **918 patient records**
- Includes **12 features** such as:
  - Age
  - Gender
  - Blood Pressure
  - Cholesterol
  - Heart Rate
  - Chest Pain Type
- Target variable:
  - `0` → No heart disease
  - `1` → Heart disease risk

---

## 🧠 Machine Learning Models Used

| Model | Description |
|------|------------|
| Logistic Regression | Simple and interpretable binary classifier |
| SVM | Effective for high-dimensional data |
| Random Forest | Ensemble method reducing overfitting |
| Gradient Boosting | Sequential ensemble improving accuracy |
| KNN | Distance-based classification |

---

## 📈 Results

| Model | Accuracy |
|------|---------|
| Gradient Boosting | **94%** ✅ |
| Random Forest | 92% |
| SVM | 87% |
| Logistic Regression | 84% |
| KNN | 81% |

👉 Gradient Boosting performed the best, showing strong predictive capability for medical datasets.
