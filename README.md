# AI-Influence-on-Healthcare-Heart-Disease-Prediction-
# ❤️ Heart Disease Prediction System

## 📌 Project Overview
The Heart Disease Prediction System is a Machine Learning project that predicts whether a person is likely to have heart disease based on medical attributes.  
The goal is to assist in early detection and support healthcare decision-making.

---

## 🚀 Technologies Used
- Python 🐍  
- Pandas & NumPy  
- Matplotlib  
- Scikit-learn  
- Jupyter Notebook  

---

## 📊 Models Implemented
- Logistic Regression  
- K-Nearest Neighbors (KNN)  
- Decision Tree  

---

## 📈 Model Performance

| Model                  | Accuracy |
|----------------------|---------|
| Logistic Regression  | 83.69%  |
| Decision Tree        | 81.88%  |
| KNN                  | 67.75%  |

✅ Logistic Regression achieved the highest accuracy.

---

## 🔍 Key Features
- Data preprocessing and cleaning  
- Train-test split (70:30 ratio)  
- Model training and evaluation  
- Confusion matrix and classification report  
- Accuracy comparison visualization  
- Model saving using Pickle
